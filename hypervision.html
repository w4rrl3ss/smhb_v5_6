HyperVision

Monitoramento avançado de hardware em tempo real com Python e PyQt5

Visão Geral

O HyperVision é um sistema de monitoramento de hardware desenvolvido em Python que fornece uma interface gráfica completa para visualização de dados de sensores do sistema, incluindo temperaturas, velocidades de ventoinhas, cargas de componentes e outros parâmetros críticos.

Interface principal do HyperVision mostrando dados de sensores e gráficos

Funcionalidades Principais

  • Monitoramento em tempo real - Atualização contínua de dados de sensores a cada 2 segundos
  • Interface gráfica intuitiva - Desenvolvida com PyQt5 para melhor experiência do usuário
  • Categorização inteligente - Organização automática de sensores por tipo e componente
  • Sistema de alertas - Cores e ícones indicam status crítico baseado em thresholds configuráveis
  • Gráficos históricos - Visualização temporal de dados com QChart
  • Filtros avançados - Capacidade de filtrar por status ou componente específico
  • Persistência de estados - Mantém preferências de visualização entre atualizações

Principais Componentes do Código

Classe HardwareMonitorGUI

Classe principal que herda de QMainWindow e gerencia toda a aplicação:

class HardwareMonitorGUI(QMainWindow): def __init__(self): super().__init__() # Estruturas de dados self.sensor_data = {} self.sensor_history = {} self.processed_sensors = set() # Configuração inicial self.initialize_hardware_monitor() self.setup_ui() # Timer para atualizações self.update_timer = QTimer() self.update_timer.timeout.connect(self.update_sensor_data) self.update_timer.start(2000)

Métodos Principais

  • initialize_hardware_monitor() - Configura a conexão com o LibreHardwareMonitor
  • setup_ui() - Cria todos os componentes da interface gráfica
  • update_sensor_data() - Atualiza os dados dos sensores
  • process_sensor() - Processa dados individuais de sensores
  • update_sensor_tree() - Atualiza a árvore de visualização
  • update_sensor_charts() - Gera/atualiza os gráficos históricos

Instalação e Uso

Pré-requisitos

  • Python 3.7 ou superior
  • Sistema Windows (compatível com LibreHardwareMonitor)
  • Bibliotecas listadas em requirements.txt

Instalação

# Clone o repositório git clone https://github.com/seu-usuario/hypervision.git cd hypervision # Crie um ambiente virtual (recomendado) python -m venv venv venv\Scripts\activate # Instale as dependências pip install -r requirements.txt

Execução

python hypervision.py
